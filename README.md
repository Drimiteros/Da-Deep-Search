# Da-Deep-Search (v.2.0) 🔎

### **Overview 🎯**
Da Deep Search allows you to locate even the deepest files in your PC, FAST!  
It's ment to be a better alternative to Windows Search.

✅ Deep file search  
✅ Fast file search
